# Legacy Gameplay Parity Gap Report

Scope: this report raises only player-visible gameplay risks. Code structure differences are ignored unless they can change legal moves, scores, turns, racks, bag count, boosts, timers, bot play, online state, notifications, recovery, or visible game understanding. `HEAD:index.html` is the legacy authority.

Verification note: `npm run test:unit` was run outside the sandbox on 2026-05-18. Gameplay parity coverage passed; the single failure was `tests/unit/logo-markup.test.js`, a non-gameplay home-logo markup assertion.
